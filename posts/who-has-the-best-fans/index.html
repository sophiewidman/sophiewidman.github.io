Who has the Best Fans in College Sports?

Author

Sophie Widman

Published

April 14, 2024

College sports fans can be crazy with a wild fan base and show their passion in many ways. One common denominator that college fans share is the joy of attending games of their favorite sports teams.

But who has the best fans with highest attendance?

Code 
library(tidyverse)
library(ggbeeswarm)
library(ggrepel)
library(scales)

volleyball <- read_csv("volleyballdata.csv") |>
mutate(sport = "Volleyball", year = 2023) 

volleyball2022 <- read_csv("volleyballdata2022.csv") |>
  mutate(sport = "Volleyball", year = 2022)

mensbball <- read_csv("mensbasketballdata.csv") |>
  mutate(sport = "Men’s Basketball", year = 2023) 

mensbball2022 <- read_csv("mensbballdata2022.csv") |>
mutate(sport = "Men's Basketball", year = 2022)

football <- read_csv("footballdata.csv") |>
mutate(sport = "Football", year = 2023) 

football2022 <- read_csv("football2022data.csv") |>
mutate(sport = "Football", year = 2022)

baseball <- read_csv("baseballdata.csv") |>
mutate(sport = "Baseball", year = 2023) 

baseball2022 <- read_csv("baseball2022data.csv") |>
  mutate(sport = "Baseball", year = 2022)

womensbball <-read_csv("womensbasketballdata.csv") |>
mutate(sport = "Women's Basketball", year = 2023)

womensbball2022 <- read_csv("womensbball2022data.csv") |>
  mutate(sport = "Women's Basketball", year = 2022)

softball <- read_csv("softballdata.csv") |>
  mutate(sport = "Softball", year = 2023) 

softball2022 <- read_csv("softball2022data.csv") |>
  mutate(sport = "Softball", year = 2022) 

sports2023 <- bind_rows(volleyball, mensbball, football, baseball, womensbball, softball)

sports2022 <- bind_rows(volleyball2022, mensbball2022, football2022, baseball2022, womensbball2022, softball2022)

sports <- bind_rows(sports2023, sports2022)
bigten <- sports2023 |> filter(Conference == "Big Ten")

powerfive <- sports2023 |> filter(Conference == "ACC" | Conference == "Big Ten" | Conference == "Pac-12" | Conference == "SEC" | Conference == "Big 12")

attendance <- sports2023 |> 
  group_by(Institution,Conference) |> 
  summarise(
    totalattendance = sum(`Accum Attendance`)
  ) |> arrange(desc(totalattendance))

nu <- attendance |> filter(Institution == "Nebraska")

lsu <- attendance |> filter(Institution == "LSU")

ggplot() + 
  geom_beeswarm(data=attendance, aes(x=totalattendance, y=Conference), color="lightgrey") + 
  geom_beeswarm(data=nu, aes(x=totalattendance, y=Conference),color="#E41C38") + 
  geom_beeswarm(data=lsu, aes(x=totalattendance, y=Conference),color="#461D7C") +
  geom_text_repel(data=nu, aes(x=totalattendance, y=Conference, label=Institution)) +
  geom_text_repel(data=lsu, aes(x=totalattendance, y=Conference, label=Institution)) + scale_x_continuous(labels=comma) + 
  labs(
    x="Total Attendance", 
    y="Conference", 
    title="Nebraska and LSU Fans Pack the Stands", 
    subtitle="Huskers and Tigers have the highest numbers of total athletic attendance in 2023-2024 seasons.", 
    caption="Source: NCAA | By Sophie Widman"
    ) +
  theme_minimal() +
  theme(
    plot.title = element_text(size = 20, face = "bold"),
    axis.title = element_text(size = 8), 
    plot.subtitle = element_text(size=10), 
    panel.grid.minor = element_blank(),
    plot.title.position = "plot"
    )

Nebraska and LSU are set apart from other schools for their attendance numbers and passionate fanbase in 2022. When looking at 2023, both teams remain at the top. Although LSU remains the lead, its numbers decrease as Nebraskas increase drastically. What did Nebraska do differently than LSU?

Code 
powerfive <- sports |> filter(Conference == "ACC" | Conference == "Big Ten" | Conference == "Pac-12" | Conference == "SEC" | Conference == "Big 12")

attendanceslope <- sports |> 
  group_by(Institution,Conference, year) |> 
  summarise(
    totalattendance = sum(`Accum Attendance`)
  ) |> arrange(desc(totalattendance))

nu5 <- attendanceslope |> filter(Institution == "Nebraska")
lsu5 <- attendanceslope |> filter(Institution == "LSU") 

ggplot() + 
  geom_line(data=attendanceslope, aes(x=year, y=totalattendance, group=Institution),color="grey") +
  geom_point(data=attendanceslope, aes(x=year, y=totalattendance, group=Institution),color="grey") +
  geom_line(data=nu5, aes(x=year, y=totalattendance, group=Institution),color="#E41C38") +
  geom_point(data=nu5, aes(x=year, y=totalattendance, group=Institution),color="#E41C38") +
geom_line(data=lsu5, aes(x=year, y=totalattendance, group=Institution),color="#461D7C") +
  geom_point(data=lsu5, aes(x=year, y=totalattendance, group=Institution),color="#461D7C") + 
  scale_x_continuous(breaks= c(2022,2023), limits = c(2021.5,2023.5)) +
  geom_text(data=nu5 |> filter(year==max(year)), aes(x=year +.15,y=totalattendance +.05, group=Institution, label=Institution)) +
  geom_text(data=nu5 |> filter(year == min(year)), aes(x=year -.15, y=totalattendance, group=Institution, label=Institution)) + 
   geom_text(data=lsu5 |> filter(year == min(year)), aes(x=year -.15, y=totalattendance, group=Institution, label=Institution)) + 
  labs(
    x="Total Attendance", 
    y="Year", 
    title="Nebraska Volleyball Day Spikes Attendance Numbers", 
    subtitle="The Husker's attendance significantly increases in one year after Nebraska Volleyball Day while LSU's attendance decreases.", 
    caption="Source: NCAA | By Sophie Widman"
    ) +
  theme_minimal() +
  theme(
    plot.title = element_text(size = 20, face = "bold"),
    axis.title = element_text(size = 8), 
    plot.subtitle = element_text(size=10), 
    panel.grid.minor = element_blank(),
    plot.title.position = "plot"
    )

Nebraska Volleyball had Nebraska Volleyball Day at Memorial Stadium. They set a world record with 92,003 fans in the stands.

Breaking it down to just volleyball, Nebraska wins by a landslide. Nebraska Volleyball Day was a massive factor to Nebraska Volleyballs attendance numbers in the 2023 season. Attendance can be an indicator of how successful athletic programs are. Although we would need more data to distinguish who has the best fan base, Nebraska’s ability to sell out a football stadium for the volleyball team shows how passionate the fan base is about their teams and speaks for itself.
